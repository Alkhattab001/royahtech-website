# Royah - Digital Strategy & AI Consulting

A modern, responsive website for Royah, a digital strategy and AI consulting company serving clients across the GCC region.

## About Royah

Royah specializes in:
- Digital Strategy & Transformation
- Process Automation & Integration
- AI Solutions & Implementation
- Business Consulting & Growth

## Features

- **Responsive Design**: Optimized for all devices and screen sizes
- **Multilingual Support**: Full Arabic and English support with RTL layout
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Contact Integration**: WhatsApp, Email, Phone, and Call Booking
- **Fast Performance**: Built with modern web technologies

## Technology Stack

- **React 18** with TypeScript
- **Tailwind CSS** for styling
- **Framer Motion** for animations
- **React i18next** for internationalization
- **Vite** for development and building
- **Lucide React** for icons

## Getting Started

### Prerequisites
- Node.js 18+ or Bun
- npm, yarn, or bun package manager

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd new-website
```

2. Install dependencies:
```bash
npm install
# or
bun install
```

3. Start the development server:
```bash
npm run dev
# or
bun run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Building for Production

```bash
npm run build
# or
bun run build
```

The production-ready files will be in the `dist` directory.
